# 信仰 Faith — WAP 单机 RPG

取材自小说《重生之贼行天下》(作者:发飙的蜗牛)中的虚拟游戏《信仰》。世界观为亚特兰大陆,以盗贼为核心、五职业、十六张地图、转职/宠物/签到/随机奇遇,纯前端、无后端、打开即玩,存档在浏览器本地。

> 本项目是受小说启发的同人玩法实现,仅作个人娱乐/学习用途,非商业、非官方。

## 玩法概览
- **5 职业**:盗贼(核心·偷窃)、战士、法师、牧师、圣骑士。每职业 5 个技能 + 一条专属职业任务链。
- **转职**:30 级一转、60 级二转,各二选一,永久属性加成 + 专属终极技能(大盗贼/影舞者、神武士、魔导师……)。
- **16 张地图**:卡罗尔大草原 → 然多湖 → 树妖林 → 纳特兰沼泽 → 绯梦平原 → 索斯山谷 → 索哥特古城 → 熔火森林 → 灰暗泥潭 → 永恒之城 → 索伦斯深渊 → 希尔顿要塞 → 黑暗死域 → 紫瞳龙穴 → 世界壁垒 → 熔火之渊,逐张解锁。
- **Boss 与套装**:树妖王、狮王卡多、巡回者狼人布索、地精寻猎者、阴影魔王、紫瞳地龙、堕落天使、熔岩巨兽……每个 Boss 掉落对应具名套装(逐火者/永夜之寂/黑血/秩序/格瑞玛神装……)。
- **装备**:6 部位、6 品质(白装/青铜/白银/黄金/暗金/传奇)、随机词缀、套装、战力对比、一键卖出。
- **成长**:升级加点(力/敏/智/体)、技能随等级解锁、洗点、满级 150。
- **宠物**:打 Boss/签到/孵蛋获得,出战提供被动属性 + 每回合协助攻击,可升级、放生。
- **每日签到**:7 天循环奖励(金币/药剂/信用点/宠物蛋)。
- **随机奇遇**:刷怪时随机触发宝箱、上锁宝箱(盗贼开锁有优势)、古老祭坛、流浪行商、精英伏击;以及精英怪随机出现。
- **剧情任务**:18 章宏大主线(围绕「秩序之章」与教皇加冕)+ 各区域支线 + 五职业专属任务链。
- **战斗**:回合制,手动/自动战斗,偷窃、暴击、吸血、中毒、击晕、减速等机制。

## 用 Vercel 静态托管(推荐)

整个项目就是一堆静态文件,无需构建。两种方式:

### A. 网页拖拽部署(最快,无需命令行)
1. 打开 https://vercel.com,用 GitHub/邮箱登录。
2. 进入 Dashboard → Add New → Project。
3. 如果走 Git:把本项目推到一个 GitHub 仓库,在 Vercel 里 Import 该仓库;Framework Preset 选 **Other**,Build Command 留空,Output Directory 留空(或填 `.`),点 Deploy。
4. 几十秒后得到一个 `https://你的项目名.vercel.app` 链接,手机扫码即玩。

### B. 命令行部署(Vercel CLI)
在本项目目录(`wapgame/`)下执行:
```bash
npm i -g vercel          # 安装一次即可
vercel login             # 按提示登录
vercel                   # 首次部署,一路回车;问 build/output 直接留空
vercel --prod            # 发布到正式域名
```
完成后终端会打印正式访问网址。

> 说明:仓库里已附带 `vercel.json`(开启 cleanUrls + 关闭强缓存,改完即时生效)。本游戏是纯静态资源,Vercel 默认就当静态站点托管,无需任何服务器配置。

## 其他分享方式
- **直接发文件**:把 `wapgame` 文件夹打包发给对方,双击 `index.html` 即可(纯 `<script>` 加载,`file://` 也能跑)。
- **GitHub Pages**:上传文件 → Settings → Pages → 选 `main` 分支根目录。
- **局域网**:`python3 -m http.server 8000`,同 WiFi 访问 `http://你的IP:8000`。

## 文件结构
- `index.html` 入口
- `data.js` 数据(职业/技能/转职/地图/怪物/Boss/装备/套装/宠物/签到/主线+职业线+支线)
- `engine.js` 引擎(属性/战斗/掉落/升级/转职/宠物/签到/奇遇/任务/存档)
- `ui.js` 界面渲染
- `main.js` 主控与自动战斗、奇遇流程
- `styles.css` 移动端暗色样式
- `vercel.json` Vercel 部署配置

## 注意
- 存档存在**各自浏览器本地**(localStorage),换设备/清缓存会丢。游戏内「系统」页可**导出/导入存档字符串**备份迁移。
- 界面按手机竖屏设计,手机浏览器体验最佳。
