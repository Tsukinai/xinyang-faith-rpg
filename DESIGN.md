# 《信仰 Faith》策划文档

单机网页回合制 RPG，取材自小说《重生之贼行天下》（作者：发飙的蜗牛）中的虚拟游戏《信仰》。纯前端（HTML/JS + localStorage），无后端、无构建。本文件整合原著考据、已实现系统、数值方案与开发路线图。

> 同人玩法实现，仅个人娱乐/学习，非商业、非官方。原著考据来源：小说全文（约 6 万行）。

---

## 1. 世界观（原著）

- **大陆**：亚特兰大陆。主城**卡罗尔城**（格林兰帝国都城，悬于卡罗尔大草原的天空之城）。
- **三大阵营**：
  - 光明·善良守序：**格林兰帝国**（人类）、**萨特恩帝国**（矮人/兽人/精灵）。
  - 黑暗·邪恶守序：**魔裔部落**、**亡灵部落**。
  - 中立：**巨人部落**。
- **信仰核心主线**：失落的神物 **秩序之章** vs **混乱之章**，各 6 卷 36 章，散落世界。集齐秩序之章成光明神殿**教皇**；集齐混乱之章成**黑暗之王**。守护者为**秩序镇守者**（龙巨混血，100 级精英）。
- **历史**：神战 → 龙族黑暗年代（龙王泽恩纳德/希伯瑞斯帝国）→ 共治之年代（秩序/混乱之章流传至今）。
- **主角**：聂言（游戏 ID 涅炎），带前世 180 级大盗贼记忆重生。
- **金句**：「孤寂的行者，追逐阴影的脚步，这是盗贼的赞歌」「信仰里没有最弱的职业，只有最弱的玩家」。

## 2. 职业体系（原著五大职业）

| 职业 | 定位 | 护甲 | 武器 | 转职链 |
|---|---|---|---|---|
| 盗贼 rogue | 敏捷物理/暴击/偷窃（主角职业） | 皮甲 | 匕首/弓 | 疾风盗贼→大盗贼/暗影盗贼→影舞者/死神 |
| 战士 warrior | 近战高生存 | 板甲 | 长剑/巨斧 | 盾甲战士/狂剑士→神武士/大剑士 |
| 法师 mage | 远程高爆发 | 布甲 | 法杖 | 元素/奥术法师→魔导师/大法师 |
| 牧师 priest | 治疗/神圣 | 布甲 | 法杖 | 圣言牧师/暗牧→神牧 |
| 圣骑士 paladin | 圣盾近战/神术 | 锁甲/板甲 | 单手剑 | 守护/禁言圣骑士→圣剑士 |

- **主属性**（DND 式）：力量/敏捷/智力/体质，每级自动成长 + 升级手动加点。
- **转职**：30 级一转、60 级二转，二选一，给永久属性 + 专属终极技能。
- **被动技能（专长）**：每职业 4 个，随等级自动领悟、常驻加成（武器专精/灵活移动/重甲精通/元素亲和等）。

## 3. 战斗系统

- 回合制，玩家与敌人按速度出手；支持手动 / 自动战斗。
- 机制：暴击、暴伤、命中/闪避、吸血、回血、中毒(DOT)、击晕、减速、偷窃（盗贼偷金币）、群体技、多段连击、斩杀(exec)、防御穿透(pen)。
- 出战宠物每回合协助攻击。
- **伤害公式**：`减伤率 = def / (def + 45 × 防御方等级)`，`伤害 = 攻击 × (1 − 减伤率) × 浮动(0.92~1.08)`。
  - 该公式使防御随等级保值、永不 100% 免伤，杜绝「防御≥攻击=0伤害」或「攻击碾压」两个极端。

---

## 4. 装备与掉落

- **品质链（原著）**：白装 → 青铜 → 白银 → 黄金 → 暗金 → 传奇。倍率 1.0/1.25/1.6/2.1/2.8/3.8；凹槽 0/0/0/2/3/5。
- **无随机词缀**：按品质模板给固定属性 + statBudget 点数分配到职业相关属性（贴近小说「防御区间 + 固定加成」写法）。
- **职业限制**：护甲类型（布/皮/锁/板）与武器类型（匕首/弓/法杖/长剑/巨斧/单手剑）按职业；项链戒指通用。不符职业无法穿戴。
- **每怪独立掉落表**：小怪主掉**材料**（蝙蝠牙齿/精丝/龙鳞…14 种）+ 少量金币，极低概率装备；精英概率更高；**Boss 大爆**——必掉多件 + 套装碎片 + 稀有材料 + 幸运宝石，10% 触发「大爆」一次掉一地（暗黑式）。
- **套装**：具名套装（黑血/逐火者/永夜之寂/秩序/格瑞玛神装…），同套 2/4 件触发递增加成。
- **Magic Find**：盗贼天赋 + 地图难度 + 精英/Boss 提升好装备掉率。

## 5. 技能书经济（原著「能刷出的技能书市场都买得到」）

- 创角仅自带 1 个入门主动技能；其余主动技能**靠技能书学习**（不再随等级自动解锁）。
- 技能书来源：**商店「技能导师」无限量购买（防卡死保底）** + 精英/Boss 掉落（额外惊喜） + 任务奖励（规划中）。
- 学习需「拥有书 + 达到等级」。转职技能由转职解锁，不走技能书。

---

## 6. 任务体系

- **主线**：18 章，围绕重生开局 → 各副本 Boss → 集齐秩序之章 → 教皇加冕。
- **职业任务链**：每职业 5 环专属任务，各具职业特色。
- **支线**：各区域可接、部分可重复。
- **隐藏任务**（规划中）：无标记、靠触发条件，高倍奖励（详见 §9）。

## 7. 地图（16 区，由易到难）

卡罗尔大草原(1) → 然多湖(4) → 树妖林(7) → 纳特兰沼泽(11) → 绯梦平原(15) → 索斯山谷(20) → 索哥特古城(26) → 熔火森林(32) → 灰暗泥潭(38) → 永恒之城(45) → 索伦斯深渊(52) → 希尔顿要塞(60) → 黑暗死域(72) → 紫瞳龙穴(88) → 世界壁垒(105) → 熔火之渊(120)。

- **怪物等级随区域固定 + 轻微浮动(±2)**；精英/Boss 额外加成。
- 解锁链：清前一张图的 Boss 解锁下一张（无 Boss 的纯刷怪图自动放行）。

## 8. 数值平衡

- **经验曲线**：`xpToNext(lv) = 50·lv^1.55 + 50·lv`，战斗经验 ×1.4 降低前期枯燥。满级 150。
- **怪物成长**：`hp = (28·lv + 5·lv^1.5)·(hpM/9)`、`atk = atkM·(3 + 0.9·lv + 0.03·lv²)`、`def = defM·(2 + 0.6·lv)`。精英 hp×2.5/atk×1.35；Boss 由高 hpM/atkM 体现。
- **难度目标**：同级裸装 5–10 回合杀小怪、会掉血、高级图偶尔阵亡；有装备显著变强。「稍微难一点」已落地。
- **经济**：金币 sink = 药水/技能书/洗点（后续 + 强化）。NPC 回收价 ≈ 市场价 ÷ 6（原著锚点）。

## 9. 进阶系统（已全部实现）

### Phase 2c — NPC 城镇 + 隐藏任务 + 声望 ✅
- **城镇 NPC hub**（「城镇」tab）：执政官声望大厅、药剂师、技能导师、拍卖行（卖材料）、炼药、铁匠锻造、个人领地、威望黑市；铁匠强化/驯兽场/转职导师指引到对应页。
- **隐藏任务**（6 个）：蝙蝠牙齿(卡迪)、郊外隐士(奥尔)、两千级楼梯(+属性点)、有灵性的战马(福尔克纳)、游魂公主、外交术(因塞莱斯)；满足触发条件(等级/击杀/Boss/声望)后现身，奖励含坐骑/属性点/宝石/声望，部分可重复。
- **声望/尊敬值**：任务+Boss 累积，5 档（陌生→政要）给 3–12% 全城购物折扣。

### Phase 3 — 深度系统 ✅
- **强化/镶嵌/潘多拉**：幸运宝石逐阶强化 +1~15（11 级起失败可碎装）；6 种宝石×5 品级镶嵌凹槽；潘多拉之盒 3 件同品质 10% 升阶。
- **生活技能**：采集（地图页）→ 炼药/锻造（城镇，配方+熟练度）；开锁（盗贼开宝箱熟练度）。4 档熟练度。
- **坐骑**：5 种（战马/座狼/狮鹫/福尔克纳/黄金地龙），驯兽场购买或 Boss/隐藏任务获得，加速度/战斗属性。
- **召唤**：由宠物系统承载（暗牧/亡灵风味体现在技能）。
- **称号系统**：8 个成就称号，佩戴给属性加成 + 顶栏显示。
- **资料片/世界事件**：主线推进开启 4 资料片（全局经验×1.2~2.0、掉率提升）；要塞保卫战(每日)+红名讨伐世界事件给威望+爆装。
- **个人领地**：40 级购置，每日税收金币+材料，可升级。
- **红名/威望**：世界事件给威望（全局货币），威望黑市兑换稀缺品。

## 10.5 借鉴的并行设计（novelgame 项目）

参考了同主题项目 `personal/novelgame` 的《游戏设计圣经》与结构化小说摘要（digest/extract_data.json）：
- **已采纳**：各职业更丰富的原著技能名单——每职业由 5 个主动技能扩充到 7 个（盗贼+切割/疾风步、战士+冲锋/战争践踏、法师+雷电术/变羊术、牧师+暗影箭/恐惧、圣骑士+审判/祈祷），覆盖控制/DOT/增益流派，落实「所有职业与盗贼同等丰富」。其城镇/NPC/训练师布局、生活职业清单、死亡惩罚、命骰等也纳入 Phase 2c/3 路线图。
- **未采纳（与本作分歧，遵从用户指令）**：
  - 该项目用「随机词缀装备（暗黑式滚词缀）」；本作按用户要求**去词缀、用品质模板固定属性**（更贴小说「防御区间+固定加成」写法）。
  - 该项目设 6 职业（含原创「猎魔者」）；本作保留**原著确证的 5 职业**（弓/弩作为盗贼专长），猎魔者列为可选扩展。

## 11. 文件结构

- `index.html` 入口
- `data.js` 数据（职业/技能/被动/转职/地图/怪物/Boss/装备/套装/材料/宠物/主线+职业线+支线）
- `engine.js` 引擎（属性/战斗/掉落/升级/转职/被动/技能书/宠物/奇遇/任务/存档）
- `ui.js` 界面渲染
- `main.js` 主控、自动战斗、奇遇流程
- `styles.css` 移动端暗色样式
- `vercel.json` 部署配置
- `DESIGN.md` 本策划文档
- `README.md` 玩法与部署说明

## 12. 部署

- 本地：`python3 -m http.server 8000` → `http://localhost:8000`
- Vercel：仓库导入（Framework 选 Other，Build/Output 留空）或 `vercel --prod`
- 存档在浏览器 localStorage，系统页可导出/导入字符串迁移。
